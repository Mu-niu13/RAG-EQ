# RAG-EQ

## Data Format
For raw data, all of them should be turned into the given format
```
{
    "topic": "...",       # eg. "credit stealing", "manager conflict"
    "query": "...",       # ended with ?
    "emotion_q": "...",   # e.g. "supportive"
    "response": "...",
    "emotion_r": "...",
}
```
